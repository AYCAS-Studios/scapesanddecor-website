Clearing Agent Application.

Application pack for customs clearing, forwarding and port agents seeking appointment to Scapes & Décor's clearing-agent panel. Complete and return to begin vetting. A formal appointment letter is issued separately to successful applicants. Scapes & Décor is a Harare-based landscaping, décor and contracting firm; current import activity includes solar street-lighting equipment shipped from China via regional ports of entry.

Return the completed application with certified copies of licences, insurance schedules and bank confirmation to agabaza@scapesanddecor.com.
